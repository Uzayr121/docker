About This Project

I created a multi-container application using Python Flask and Redis.

Project Features:

  • A Flask web app with two routes:
    • /: Displays a welcome message.
    • /count: Increments and displays a visit count stored in Redis.
  • Redis is used as a key-value store to track the visit count.
  • Both Flask and Redis services are dockerized using Docker Compose.
  • Persistent storage is configured for Redis using volumes.
  • Environment variables are used to manage Redis connection details.
  • The Flask service is scalable, and Nginx is used as a load balancer.

Testing the Application:

  • Visit http://localhost:5000 to see the welcome page.
  • Visit http://localhost:5000/count to see the visit count increase with each refresh.

Bonus Features:

  • Redis data persistence using Docker volumes.
  • Environment variables for flexible configurations.
  • Load balancing with Nginx and multiple Flask instances.
